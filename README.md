# QZhi Othello

## 构建工具

Visual Basic 6.0 SP6

## 产品简介

一个简单的翻转棋实现。

## 安装步骤

本产品为“便携式”应用程序，若要安装，请打开 Othello.vbp 自行编译，或下载 Release 中已编译好的二进制文件。

## 使用方法

参见 /docs/help/help.html
